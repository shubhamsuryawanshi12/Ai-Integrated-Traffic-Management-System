## Presentation Tips

### Design Guidelines
1. **Consistency**: Use the same font, colors, and style throughout
2. **Readability**: Font size minimum 24pt for body text, 36pt+ for titles
3. **Visuals**: Use images, diagrams, and charts to break up text
4. **White Space**: Don't overcrowd slides
5. **Color Scheme**: Use professional, contrasting colors
6. **Animations**: Use sparingly and purposefully

### Content Guidelines
1. **Rule of Thirds**: Maximum 3 main points per slide
2. **6x6 Rule**: No more than 6 bullets with 6 words each
3. **One Message**: Each slide should have one main message
4. **Visual > Text**: Show, don't just tell
5. **Consistency**: Maintain same structure across similar slides

### Delivery Tips
1. **Practice**: Rehearse multiple times
2. **Timing**: Stay within time limit (usually 15-20 minutes)
3. **Eye Contact**: Look at audience, not slides
4. **Voice**: Speak clearly and at moderate pace
5. **Backup**: Have contingency for technical issues
6. **Demo**: Practice demo multiple times
7. **Questions**: Prepare for Q&A session

### Technical Preparation
1. Test all equipment beforehand
2. Have backup of presentation (USB, cloud, email)
3. Test demo on presentation computer
4. Prepare demo video as backup
5. Check audio/video if using multimedia
6. Arrive early to set up
